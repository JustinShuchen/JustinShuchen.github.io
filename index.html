Projects

LLM-Based Embodied Task Planning

Multi-modal understanding of industrial tasks is crucial for intelligent operations. This project presents an end-to-end multi-modal data generation and task planning pipeline covering 10 representative tasks. The pipeline integrates real and AI-generated images, YOLO-World-based auto-annotation, and GPT-driven instruction generation into a unified JSON schema. Built on Qwen2.5-VL-78 with LoRA fine-tuning, the system produces image-instruction-plan triplets and enables efficient reasoning. Experiments demonstrate strong consistency and generalization, providing effective support for multi-modal intelligence in industrial applications.

Key contributions of this project include:

  • Design logical instructions for actions.
  • Use ChatGPT to generate images for training.
  • Develop a task planning agent based on Large Language Models.
